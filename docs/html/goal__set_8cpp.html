This function manages the goal_set node.
It does the ROS init, set the NodeHandle to access the comunication with the ROS system, and creates an action client to "/reaching_goal".
It manages the option menu where the user can set the goal coordinates, delate the current goal set, get the number of reached and deleted goals, and exit the menu.
